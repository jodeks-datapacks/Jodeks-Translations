# Jodek's Translations

A library of translation keys for my datapacks.

Contributions are welcome to help expand language support!

## How to Contribute

1. **Fork the repository**
   - Click the "Fork" button on the repository page

2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/JodeksTranslations.git
   ```

3. **Make your changes**
   - Edit existing translations, or
   - Add translations for new languages by creating a new file from the table below
   - **Tip:** Copy `en_us.json` as a starting point — it contains all translation keys

4. **Commit and push**
   ```bash
   git add .
   git commit -m "Add [language] translations"
   git push origin main
   ```

5. **Create a pull request**
   - Go to your fork on GitHub and click "New Pull Request"

I'll review and merge approved contributions in the next release.
